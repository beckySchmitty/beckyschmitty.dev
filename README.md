# beckyschmitty.dev

Personal portfolio website for Becky Schmitthenner.

Built with Vite + React + TypeScript.

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is deployed via Vercel at [beckyschmitty.dev](https://beckyschmitty.dev)
